# Security

Do not commit API keys, keystores, signing passwords, database files, recordings, transcripts, or sidecars containing real conversations.

If a transcription credential is exposed, revoke it with the provider and save the replacement through the app's Settings screen. The app stores provider credentials with Android Keystore-backed AES-GCM encryption and disables Android application backup.

For a vulnerability report, contact the repository owner privately rather than opening a public issue containing sensitive data.
